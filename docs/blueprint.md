# **App Name**: MailPlus Outbound Leads CRM

## Core Features:

- NetSuite Lead Sync: Sync leads from NetSuite via API to populate the CRM.
- Lead List: Display a list of synced leads with key information like name, company, and status.
- Lead Profile: Detailed lead profiles showing all available information and activity history.
- Direct Call: Initiate calls to leads directly from their profile using the AirCall API.
- Call Activity Log: Log call activities automatically after each call via AirCall, noting time, duration, and status.
- AI Lead Scoring: AI-powered lead scoring to rank leads by potential value, displayed prominently in the lead list.
- Talking Point Suggestion: Utilize AI tool to generate suggested talking points based on the lead's profile to aid in personalized conversations.

## Style Guidelines:

- Primary color: Professional blue (#095c7b), reflecting trust and stability, for the main interface elements.
- Background color: Light blue (#d0dfcd), creating a clean and open feel.
- Accent color: Vibrant orange (#eaf143) to highlight important actions and calls-to-action.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look that will offer great legibility and usability in an application.
- Use a set of clean, consistent icons to represent lead status, activity types, and actions.
- Emphasize a clear hierarchy and intuitive navigation for efficient lead management.
- Use subtle transitions and animations to provide feedback on user interactions.
