# Build Project

Follow these steps to install dependencies and build the Next.js application.

## 1. Verify Node.js Environment
Ensure Node.js version is 20 or higher.
```bash
node --version
```

## 2. Install Dependencies
Install all required npm packages.
```bash
npm install
```

## 3. Typecheck (Optional but Recommended)
Run TypeScript static analysis to catch potential errors.
```bash
npm run typecheck
```

## 4. Build the Application
Compile the Next.js application for production.
```bash
npm run build
```
