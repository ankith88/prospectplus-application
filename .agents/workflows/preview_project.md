# Preview Project

Follow these steps to start the development server and preview the application.

## 1. Start Development Server
Run the Next.js development server on port 9002.
```bash
npm run dev
```

## 2. Access the Application
Open your browser and navigate to the following URL:
[http://localhost:9002](http://localhost:9002)

## 3. Verify Functionality
Ensure the page loads and the "ProspectPlus" title is visible in the browser tab.
